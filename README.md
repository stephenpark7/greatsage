# GreatSage

This monorepo contains both the client and server codebases for the GreatSage project.

## Directory Structure

    greatsage/
        client/  # Client-side code
            src/
            public/
            package.json
            ...
        server/  # Server-side code
            src/
            config/
            package.json
            ...

## Instructions

To run the server: `docker-compose up -d`
